Class: ngeoModal

ngeoModal

new ngeoModal($parse) → {angular.Directive}

Provides the "ngeoModal" directive.

This directive shows a Bootstrap modal when the ngModel expression evaluates to true, and it hides it when the ngModel expression evaluates to false.

The directives also changes the ngModel value when the user manually closes the modal.

This directive is based on Bootstrap's modal classes and associated jQuery plugin.

  </div>

</ngeo-modal>

Note: for z-indexing purpose, the modal DOM element is automatically moved
to document body element.
